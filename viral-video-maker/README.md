# Viral Video Maker SaaS

🎥 **Plataforma SaaS completa para criação automática de vídeos virais otimizados para YouTube e TikTok.**

## 🚀 Features

### Frontend React
- ✅ **React 18** com TypeScript
- ✅ **Vite** para build rápido
- ✅ **Tailwind CSS** para estilização
- ✅ **Framer Motion** para animações
- ✅ **React Router** para navegação
- ✅ **Design responsivo** (mobile-first)

### Sistema Completo
- ✅ **Autenticação** (login/register)
- ✅ **Dashboard** com estatísticas
- ✅ **Upload de vídeos** com drag & drop
- ✅ **Gestão de vídeos** (CRUD)
- ✅ **Interface moderna e profissional**

### Backend & Infra
- ✅ **Express.js** API server
- ✅ **Orchestrator** com 8 agentes
- ✅ **CI/CD** pipeline configurado
- ✅ **Netlify** deployment ready
- ✅ **Docker** support

## 🚀 Começar

### Pré-requisitos
- Node.js 18+
- npm ou yarn
- Git

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/viral-video-maker.git
cd viral-video-maker
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o servidor:
```bash
npm run dev
```

4. Acesse: http://localhost:3000/

### Environment Variables
